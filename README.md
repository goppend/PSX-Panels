# PSX Panel Recreations

[![CC BY-NC-SA 4.0][cc-by-nc-sa-shield]][cc-by-nc-sa]

These are recreations of various plastic panels found on the Sony PSX DVR.
Some small changes were made to improve durability and 3D printing friendliness. E.g. removing thin features on the interior, and making the mating features more solid.

FAQ:
Q: I have model DESR-XXXX PSX, which files should I use?
A: There were two generations of PSX , PSX1 and PSX2, each of which had a few models. The model number breakdown is [here](https://www.psdevwiki.com/ps2/PSX#Models).

Q: What software do I need to modify the source files?
A: The .SLDPRT files were made with Solidworks Design, which is a lower cost maker version of the enterprise CAD tool. The STEP files should be able to be imported into most 3D CAD tools e.g. Fusion 360.

Q: Can you please help me with a fitment issue?
A: Feel free to create an issue with what's wrong, including as much detail and pictures if possible and I'll do what I can when I have the time. I have tested the PSX1 parts in my DESR-7000 so I know for sure that they fit. I do not own a PSX2 so I can't actually test those parts myself.
